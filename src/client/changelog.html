  • I noticed that there was an odd thing with phasing when you went super fast using the booster, so to fix that you have to change the this.topSpeed line in server to this.topSpeed = (c.runSpeed * (this.SPEED * 4.25) * this.skill.mob / speedReduce) / 2.75; and you have to set the accel lines in advancedcollide to my.accel.x += modifiers._me * force.x * 2; my.accel.y += modifiers._me * force.y * 2; n.accel.x -= modifiers._n * force.x * 2; n.accel.y -= modifiers._n * force.y * 2; that should fix all of it, enjoy!
